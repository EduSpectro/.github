# 🌈 Welcome to EduSpectro! ✨

### 「光をあやつり、自然のナゾを解き明かそう！」
EduSpectro（エデュ・スペクトロ）へようこそ！  
私たちは、最新のマイコンや3Dプリンタを使って、誰もが自分だけの「分光器（光を分ける装置）」を作れる未来を目指しているプロジェクトです。

---

## 🚀 はじめての方へ（Getting Started）
どこから始めていいか迷ったら、このステップをチェック！

1. **[📖 総合ガイドを見る]**  
   まずは分光器で何ができるのか、どんな材料が必要かを知ろう！
2. **[🛠️ 装置を組み立てる (Station & 4X)]**  
   3Dプリンタでケースを作り、RP2350マイコンをセットしよう。
3. **[📊 アプリを動かす (Lab)]**  
   パソコンにアプリをダウンロードして、光を測ってみよう！

---

## 📂 プロジェクトのなかみ
EduSpectroは3つのパーツでできています。

* **⚙️ EduSpectro 4X** ：分光器の「脳」となるプログラムと回路。
* **💻 EduSpectro Lab** ：グラフを表示して、データを分析するアプリ。
* **📦 EduSpectro Station**：光を集める「体」となる、3Dプリント用データ。
